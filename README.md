🌸 SafeStep

Empowering women through smart safety tech
SafeStep is a web-based women safety platform designed to provide instant emergency support and digital protection tools. The system integrates SOS alerts, live location tracking, safety timers, fake call simulation, and emergency contact management into a simple and responsive interface.

🚀 FEATURES

🚨 Emergency SOS Button
Instantly triggers an emergency alert and fetches live location.
📍 Live GPS Location Sharing
Generates a Google Maps link using the browser Geolocation API.
⏳ Safety Timer
Automatically triggers SOS if the timer is not cancelled in time.
📞 Fake Call Simulation
Creates a simulated incoming call to exit uncomfortable situations.
👩‍👩‍👧 Emergency Contact Storage
Saves emergency contact details using browser local storage.
💻 Responsive Design
Works smoothly on desktop and mobile devices.
🛠️ Technologies Used
HTML5
CSS3
JavaScript
Browser Geolocation API
Local Storage

⚙️ How It Works

User logs in with basic details.
Accesses the safety dashboard.
Can trigger SOS, share live location, set safety timers, or simulate fake calls.
If a safety timer expires, SOS activates automatically.
Emergency contact information is stored for quick access.
SafeStep acts as a digital safety companion, combining real-time tracking and automated alerts into one accessible platform.
